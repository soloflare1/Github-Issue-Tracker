## Difference between `var` , `let` &  `const`
* `var` - can redeclare, function scope.
* `let` -  can reassign, block scope, can't redeclare.
* `const` - can't reassign, block scope, can't redeclare.
  
## Spread operator (...)
* It expands/copies objects/arrays.

## Difference between `map()`, `filter()`, and `forEach()`
* `map()` - Modify each item in array & return new array.
* `filter()` - Select items based on condition & return new array. 
* `forEach()` - Loop through each items & no return.

## Arrow function
* `const variableName = () => {}`
* A shorter way to write a function

## Template literals
* It use backticks & ${} to insert variables
* Can write multi-line string/HTML inside
